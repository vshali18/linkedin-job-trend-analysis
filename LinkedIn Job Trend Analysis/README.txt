# LinkedIn Job Trend Analysis Using Python and Power BI

## Project Overview

The **LinkedIn Job Trend Analysis** project analyzes job posting data to identify hiring trends across companies, locations, work types, experience levels, and salary distributions. The project uses Python for data preprocessing and Power BI for creating an interactive dashboard that helps visualize employment trends.

---

## Problem Statement

Job seekers often find it difficult to analyze thousands of job postings to understand current hiring trends, salary patterns, preferred locations, and employment opportunities. This project simplifies the process by analyzing LinkedIn job data and presenting the results through an interactive dashboard.

---

## Objectives

* Analyze LinkedIn job posting data.
* Identify top hiring companies.
* Analyze job opportunities by location.
* Study work type distribution.
* Analyze experience level requirements.
* Visualize salary distribution.
* Build an interactive Power BI dashboard.

## Key Insights

* The dataset contains over **1.8 million job postings**.
* More than **1.2 million companies** are represented.
* Full-time positions account for the majority of job postings.
* Mid-Senior level roles dominate the hiring market.
* Major cities show higher recruitment activity.
* Salary varies significantly across different positions.
